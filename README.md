# Dimitrios Antoniadis Portfolio

Personal portfolio for Dimitrios Antoniadis, built with React and Vite.

## Stack

- React
- Vite
- Oxlint
- Custom CSS

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## Notes

The site is designed as a concise developer portfolio: production work first, clear contact paths, responsive layouts, and a dark technical visual system with restrained cyan accents.
